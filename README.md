# Simple-3-Body-Orbital-Mechanics
Physics simulation of a three-body system with 2 planets orbiting a star at a fixed origin.

# Orbital Mechanics Simulator
Simulates 2 planets orbiting a star.

# Physics
Models a gravational force exerted on both planets by star and by each other using Newton's Law of Gravatation 

# How to Run
Run in any python environment with matplotlib installed.

# Inputs
Mass in kilograms
Velocities in m/s
(Make star mass much larger and avoid coordinates close together to avoid massive gravatational force)

# Outputs
Plots comparing orbits of both planets about star at fixed origin.
